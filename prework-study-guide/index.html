HTML

  • HTML = Hypertext Markup Language
  • The head element contains information or metadata, that helps the browser to render the page correctly. Elements included within the head are not visible when a user looks at a webpage.
  • The following metadata tags are found in the head: title, style, base, link, meta, and script.
  • The body element represents the visible content shown to the user and is an example of a semantic HTML element.
  • In semantic HTML elements, the name of the element describes its content and the role it plays on a webpage.
  • The following metadata tags are found in the body: h1, h2, h3, etc., p, ul, ol, li, img, br, and a.

CSS

  • CSS = Cascading Style Sheets
  • Defines how webpage content is presented to users (colors, aesthetics, visual layout, and organization)
  • Inline CSS = quick and permanent but less flexible change directly in the HTML file i.e. style attribute
  • Internal CSS Style Sheet = Contains CSS rules in the head section of the HTML file.
  • External CSS Style Sheet = Most common; Allows CSS ruels to be in a separate file; linked in the HTML's head element using a link element
  • Selector - defines the element or attributes to which the rules, or declarations, will apply.
  • Declarations - contain the CSS property we want to apply and the value of the property
  • Margin - indicates how much space we want around the outside of an element.
  • Padding - indicates how much space we want around the content inside an element.
